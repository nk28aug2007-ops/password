# password
To identify whether a password is strong or weak based on security rules and educate users on creating safer passwords.  🧠 Key Concept  A strong password is hard To guess and resistant to attacks like brute force or dictionary attacks, while a weak password is easy to crack.
Idea:
Check whether a password is weak, medium, or strong
🔧 Features:
Length check
Uppercase/lowercase
Numbers & special characters
